Estilos css

Selectores

  • div

    Sirve para seleccionar una etiqueta específica.

  • *

    Sirve para seleccionar todos los elementos

  • .class

    Sirve para selecconar un elemento que tenga una clase específica.

  • #id

    Sirve para selecconar un elemento que tenga un id específico.

  • div p

    Sirve para seleccionar a todos los hijos que sean de una etiqueta específica, dentro de un elemento en específico.

  • div, p

    Sirve para seleccionar más de un elemento en específico.

  • div > p

    Sirve para seleccionar el hijo directo de un elemento en específico.

  • div + p

    Sirve para seleccionar la primera etiqueta que este al lado de una etiqueta específica.

  • div ~ p

Propiedades

  • color: #FFFFFF

    Sirve para poner un color a los textos.

  • font-size: 24px

    Sirve para dar tamaño a las fuentes de texto.

  • font-weight: 700

    Sirve para dar grosor a las fuentes de texto.
